# Agrilink

Plataforma B2B para conectar compradores, agentes e fornecedores do setor agro em Angola.

## Stack

- React + TypeScript + Vite
- Supabase (Auth + Database)
- Tailwind CSS + componentes UI

## Ambiente local

1. Instalar dependências:

```bash
npm install
```

2. Iniciar em desenvolvimento:

```bash
npm run dev
```

3. Gerar build de produção:

```bash
npm run build
```

## Domínio de produção

- https://agrilink.ao

## Fluxos principais

- Autenticação por email/senha
- Login com Google com onboarding em etapas
- Gestão de perfil e tipos de utilizador
- Publicação e descoberta de produtos
